## 🚀 部署步骤

### 1. 创建GitHub仓库
- 仓库名：`order-query`
- 设置为公开仓库

### 2. 上传文件
- 上传 `index.html` 到仓库根目录
- 确保文件名为 `index.html`

### 3. 启用GitHub Pages
- 进入仓库设置
- 找到 "Pages" 设置
- 选择 `main` 分支
- 选择 `/ (root)` 文件夹
- 保存设置

### 4. 访问网站
- 等待几分钟部署
- 访问：`https://你的用户名.github.io/order-query`
